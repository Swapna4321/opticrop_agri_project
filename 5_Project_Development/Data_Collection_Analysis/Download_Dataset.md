Dataset used for crop recommendation system contains soil nutrients and environmental features like N, P, K, temperature, humidity, pH and rainfall.
Popular open-source agricultural datasets are available through platforms such as Kaggle and the UCI Machine Learning Repository. For this project, the Crop_recommendation.csv dataset is used, which is downloaded from Kaggle for agricultural prediction and crop recommendation analysis. Use the link below to access the dataset:

Dataset Link: https://www.kaggle.com/datasets/chitrakumari25/smart-agricultural-production-optimizing-engine

Note: There are multiple techniques available for understanding and analyzing agricultural data. This project uses a representative agricultural dataset and selected analytical methods for crop recommendation and smart farming prediction workflows. Additional visualization and analysis techniques may also be implemented as required.
