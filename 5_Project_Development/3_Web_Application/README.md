# OptiCrop Web Application

## Overview

The Web Application module provides a user-friendly interface for predicting suitable crops based on soil nutrients and environmental conditions.

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Machine Learning

## Input Parameters

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH Value
- Rainfall

## Features

- User-friendly interface
- Crop recommendation system
- Real-time prediction
- Easy data input

## Files Included

- app.py
- templates/index.html
- requirements.txt
- Web_Application_Documentation.docx

## Output

The application predicts and displays the most suitable crop for cultivation based on the provided input parameters.

## Conclusion

This web application helps users obtain crop recommendations quickly and efficiently, supporting smart agriculture and better farming decisions.
