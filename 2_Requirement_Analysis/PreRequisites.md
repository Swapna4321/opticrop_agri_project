# Pre-Requisites

## Software Requirements

### Programming Language
- Python 3.x

### Development Tools
- Visual Studio Code
- Jupyter Notebook

### Libraries
- Pandas
- NumPy
- Scikit-Learn
- Flask
- Matplotlib
- Seaborn

### Version Control
- Git
- GitHub

## Hardware Requirements

- Processor: Intel i3 or above
- RAM: 4 GB or higher
- Storage: 10 GB available space

## Technical Knowledge

- Python Programming
- Machine Learning Basics
- Data Analysis
- HTML and CSS
- Flask Framework

## Dataset Requirements

The dataset should contain:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall
- Crop Label
