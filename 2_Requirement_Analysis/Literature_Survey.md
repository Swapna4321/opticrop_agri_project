# Literature Survey

## Overview

Recent advancements in Machine Learning have enabled the development of intelligent agricultural systems for crop prediction and recommendation.

## Existing Approaches

Various studies have applied machine learning algorithms such as:

- Decision Trees
- Random Forest
- K-Means Clustering
- Logistic Regression
- Support Vector Machines

These models analyze soil nutrients and environmental factors to recommend suitable crops.

## Findings

- Machine Learning improves crop recommendation accuracy.
- Data-driven decisions enhance agricultural productivity.
- Intelligent systems reduce crop selection risks.
- Predictive analytics supports sustainable farming.

## Conclusion

The literature review highlights the effectiveness of machine learning techniques in agriculture and supports the development of the OptiCrop recommendation system.
